> A circular image representing one person

## Treatment

- Shape: _circle_
- Dimensions: _height == width_
- Border: _optional_
  - excluded: _1.25rem_
  - color: _variable_ (default: white)
  - position: _inside_
  - size: _1px:1rem_ (rounded up for browser support)
    - min: _2px_
    - max: _5px_

## Custom Properties

```css
--Avatar--size
```
