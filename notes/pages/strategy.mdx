# Strategy

## One function

All components will be are simple wrappers around a single class-resolution function.

`@planningcenter/utilities`

```js
export function mapPropsToClassNames({
  block = "",
  element = "",
  modifierName = "",
  modifierValue = ""
} = {}) {
  function getModifier(name, value) {
    if (!name || !value) {
      return;
    }

    if (value === true) {
      return `--${name}`;
    }

    return `--${name}_${value}`;
  }

  return [
    block && `${block}`,
    element && `__${element}`,
    getModifier(modifierName, modifierValue)
  ].join("");
}
```

Media queries complicate this a little so here is a target function for getting classnames from propse.
But classes all get resolved via this function.
Understand this function and you understand the tooling.
